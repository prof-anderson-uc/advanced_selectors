Attribute Selector Syntax

element[attr="value"]
Select an element that has this attribute
element[attr^="value"]
Select an element based on how the attribute begins
element[attr$="value"]
Select an element based on how the atrtibute ends
element[attr*="value"]
Select an element if the attribute contains this value anywhere

Pseudo Element Syntax

element::before
Put something to the left of an element
element::after
Put something to the right of element
element::first-letter
Select only the first letter of an element
element::first-line
Select only the first line of text

Pseudo Class Syntax

element:hover
Only select an element while the mouse is hovering over it
element:first-of-type
Select only the first of a kind of element
element:last-of-type
Select only the last of a kind of element
element:nth-child()
Selects sibling child elements based on what's inside the (). Can include keywords like odd or a specific number
element:is()
Selects any element that matches any of the selectors listed in the (). The list is forgiving.
element:not()
The negation, or matches-none, pseudo-class selects any element that is NOT represented in the ().
element:has()
Selects an element if any of the child elements inside it contain what is inside the ().
